# Project: Formula 1 Race Win Prediction Using Machine Learning

## 📝 Objective
Develop an immersive, stunning, and highly accurate AI system that predicts the probability of an F1 driver winning a race using historical race data from the `FastF1` library. The system will feature a **premium 3D scrollable web interface** built with Three.js (React Three Fiber) to visualize race data, ML predictions, and car telemetry.

## 🚀 Key Features
1. **Machine Learning Backend (Python, FastF1, scikit-learn, XGBoost)**
   - Extract lap times, telemetry, weather, qualifying data, and race context.
   - Train and compare Logistic Regression, Gradient Boosted Trees (XGBoost/LightGBM), and Random Forests.
   - Analyze feature importance (e.g., qualifying pos, track temperature, gap to pole).
2. **Immersive 3D Frontend (Next.js, Three.js, React Three Fiber, GSAP)**
   - **Scroll-based 3D animations**: As the user scrolls, a high-quality 3D F1 car model navigates around a stylized race track.
   - **Dynamic UI**: Glassmorphism, smooth micro-animations, curated neon/dark mode color palettes (inspired by actual F1 telemetry interfaces).
   - **Interactive Data Visualization**: 3D charts, telemetry graphs, and prediction scorecards that interact with the 3D environment.
